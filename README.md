# Web-Enabled-Database-Tutorial
The web enabled database tutorial is built in PHP and hosted via XAMMP. The website exists as “PHP-MySQL-Tutorials” with fully implemented CSS. Another version exists as “PHP-MySQL-Tutorials-minCSS” with minimum CSS intended to only manage the basic layout of the websites content.

**Index/Landing Page:**
Displays the website overview.
The goal of this website is to demonstrate how to utilize PHP to connect a website to a MySQL Database. This website will also demonstrate how to perform CRUD operations and to also generate various types of graphs based on data drawn from the database.
Provides links and summaries to other pages within the website.

**PHP Form Tutorial Page:**
This page is a tutorial that demonstrates the process of creating a form using PHP code. The form on this page does NOT utilize MySQL or connect to a database. This page illustrates the steps required to successfully create a PHP form and allows users to interact with a pre-built example form.

**PHP MySQL Tutorial(s) Page:**
This page illustrates the sequence of the PHP MySQL Tutorials from W3Schools. This page also provides links to three detailed and interactive PHP MySQL lessons. How to insert data into a database using a form, updating data in a database using a form, and displaying data returned via a query in a database.

**Inserting Data into a Database Using a Form Page:**
This page illustrates how to incorporate an input form onto a PHP page that allows the user to enter data into the database. The form on this page DOES utilize MySQL and connects to a database. This page illustrates how to incorporate an input form onto a PHP page that allows the user to UPDATE data within the database. The form on this page DOES utilize MySQL and connects to a database. This page provides the steps required to successfully update the user’s previous entry.

**Data Visualization Page:**
This page illustrates how to generate three types of dynamic graphs using data drawn from a database. There are examples of and instructions for the following charts: Pie Chart, Column Chart, and Spider/Radar Chart.
